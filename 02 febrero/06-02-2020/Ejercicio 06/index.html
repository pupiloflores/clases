1. Si un h1 tiene un selector de clase "normal", el texto tendrá un tamaño de 10px.

  2. Si la etiqueta span está contenida dentro de otra etiqueta span, el texto aparecerá de color amarillo.

  3. Si la etiqueta p contiene la clase "azul" y "amarillo" al mismo tiempo, el texto deberá pintarse de color verde.

  4. Si la etiqueta span contiene la clase "azul" y "amarillo" al mismo tiempo, el texto deberá ser de tamaño 50px.
